Krono — MetaModule plugin **v1.0.1** (faceplate compatibility fix).

This patch release keeps the embedded VCV manifest at **2.1.0** and updates MetaModule faceplate handling so the panel is correctly discovered at runtime.

## Highlights

* Added MetaModule-specific `setPanel(Svg::load(... "res/KronoPanel.svg"))` in `KronoWidget` (without changing desktop VCV behavior).
* Moved panel asset to the required package path (`assets/KronoPanel.png`, not `assets/res/KronoPanel.png`).
* Brand remains **Jolin**; module slug remains **Krono**.

## Hardware testing

**Not verified on physical MetaModule hardware** (hardware unavailable). Behavior on device is unconfirmed. Please report issues at [krono_metamodule issues](https://github.com/sitar93/krono_metamodule/issues).
