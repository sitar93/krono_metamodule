Krono — MetaModule plugin **v1.0.0** (packaging release).

This is the first **MetaModule** distribution of Krono. The embedded VCV manifest (`plugin.json`) remains **2.1.0** (same module sources as [krono_vcv](https://github.com/sitar93/krono_vcv) v2.1.0). The **v1.0.0** tag here refers to this `.mmplugin` release line only.

## Highlights

* **Krono.mmplugin** — install on a compatible [MetaModule](https://www.4mscompany.com/metamodule.html) per 4ms documentation
* Module behavior aligned with [KRONO Eurorack firmware](https://github.com/sitar93/krono) and the [VCV Rack port](https://github.com/sitar93/krono_vcv)
* Brand **Jolin**; module slug **Krono**

## Hardware testing

**Not verified on physical MetaModule hardware** (hardware unavailable). Behavior on device is unconfirmed. Please report issues at [krono_metamodule issues](https://github.com/sitar93/krono_metamodule/issues).
